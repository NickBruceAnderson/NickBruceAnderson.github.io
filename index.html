Data Wrangling, Cleaning, Exploration

I work data science efforts on a project basis, and after taking the first step of aligning on a business problem to solve, I begin the data cleaning and exploration steps that are key to garnering successful insights, predictions, and overall - projects.

Here's how:

My Data Skills & Tools

  • Querying Softwares:
    SQL, NOSQL, PostgreSQL
  • Data Navigation fundamentals:
    SELECT, FROM, WHERE, ORDER BY, TOP
  • Aggregate functions and viewing:
    SUM, COUNT, GROUP BY, HAVING
  • Window functions to pass data:
    PARTITION BY
  • Joining datasets:
    Proper primary-foreign key relationships, INNER JOIN, (and sometimes) OUTER/LEFT/RIGHT JOIN
  • Cleaning Mantra: Garbage-in = Garbage-out
  • Functions in Cleaning:
    CASE-WHEN-ELSE, CAST, CONVERT, UPDATE, ALTER, DROP
  • Techniques for Cleaning:
    field standardization, NULL imputation and/or dropping, string parsing, dupe-removal, clean feature selection
  • Coding Skills
    python: numpy, pandas, matplotlib, snsplot and R: dplyr, ggplot
  • Advanced Querying via:
    common table expressions (CTEs), nested queries, temp tables
